# Kyhl's New Fruit: Meats in Disguise

A parody landing page for "New Fruit," a revolutionary (and fictional) product line where common orchard fruits are actually meticulously disguised high-density animal proteins.

## Features

- **Surprise! It's Ham**: A bold, high-conversion landing page.
- **The Science of Meatosis™**: Detailed explanations of "Cellular Trans-Fructification."
- **Interactive Game**: A "Fruit or Meat?" challenge where the answer is always meat.
- **Audio Experience**: Includes the "New Fruit" theme song.
- **Responsive Design**: Built with Tailwind CSS for a modern, mobile-friendly look.

## Assets

- `apple_steak.png`, `banana_sausage.png`, etc.: Product "photography."
- `theme.mp3`: The official brand anthem.
- `index.html`: Single-page application logic and styling.

## Technical Details

- **Frontend**: HTML5, Tailwind CSS (via CDN).
- **Interactivity**: Vanilla JavaScript for the game logic and audio controls.
- **Fonts**: Google Fonts (Inter, Merriweather, Titan One).
