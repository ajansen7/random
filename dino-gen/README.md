# Toddler Dino Creator

A Streamlit-based application designed for kids to build their own custom dinosaurs.

## Features

- **Select a Base Dino**: Choose from T-Rex, Triceratops, Stegosaurus, and more.
- **Accessorize**: Add party hats, sunglasses, superhero capes, or even astronaut helmets.
- **Set the Scene**: Place your dino in a Realistic Jungle, Space, or a Modern City.
- **Choose an Art Style**: Options include Photorealistic (Pixar style) and Cartoon.

## Technical Details

- **Frontend**: Built with [Streamlit](https://streamlit.io/).
- **AI Integration**: Designed to be used with AI image generation APIs (like OpenAI DALL-E).
- **Prompt Engineering**: Automatically constructs detailed prompts based on user selections to ensure high-quality results.

## Usage

1. Install requirements: `pip install streamlit`
2. Run the app: `streamlit run dino_app.py`
